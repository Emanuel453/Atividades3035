Sobre este projeto

Este documento reúne, em um único arquivo, HTML, os principais conceitos estudados no curso: elementos de texto, tabelas, formulários,mídias,atributos e boas práticas de acessibilidade.

Trecho com texto em linha destacado usando a tag <span>

Oque esse projeto cobre

  • Estrutura Básica do HTML (doctype,head,body)
  • Tags estruturais semânticas
  • Formatação e hierarquia de textos
  • Tabela acessível com scope
  • Formulário completo com label fieldset e output
  • Imagem,vídeo e aúdio

Ordem de estudo sugerida

  1. Estrutura Básica
  2. Tags estruturais
  3. Elementos de texto
  4. Tabelas e formulários
  5. Mídias e acessibilidade
